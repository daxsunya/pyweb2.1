# Локальный запуск
```
python -m pip install --upgrade pip
npm install mini-ssg
npm run build
```
# Шаги по развертыванию
- Установить необходимые модули
```
python -m pip install --upgrade pip
npm install mini-ssg
```
- Собрать проект
```
npm run build
```
- Запушить в pages (actions)
- Выбрать целевую ветку для отображения сайта (settings)
